===升级方法=== 

------------------------------------- 

1. 覆盖上传所有文件 。

2. 删除 data/mysqli_error_trace.php 文件

3. 删除 data/cron.cache.php 文件，如使用了定时任务自动版，需重新在后台设置提交一次。

4. 更新缓存。 



【提示】海洋CMS升级必须逐个版本升级，不可跨版本。 

例如：8.1->8.2············->9.94->9.95
